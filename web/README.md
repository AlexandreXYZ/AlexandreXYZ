Lightweight web frontend for the AlexandreXYZ project.

## Features

- Minimal scaffolding for a modern web app
- Development and production build scripts

## Prerequisites

- Node.js (16+)
- pnpm

## Install

```bash
pnpm install
```

## Development

Start a development server with hot reload:

```bash
pnpm dev
```

## Build

Create an optimized production build:

```bash
pnpm build
```

## Preview

Serve the production build locally:

```bash
pnpm preview
```
